1. react-router

为 React而生的声明式路由。

1.1. 安装

使用 npm:

$ npm install --save react-router

注意: 这个包提供了React Router的核心路由功能，但是您可能不想直接安装它。如果您正在编写一个将在浏览器中运行的应用程序，那么您应该安装react-router-dom。类似地，如果您正在编写一个React Native应用程序，则应该安装react-router-native。这两种设备都将安装react-router作为其依赖项。

然后用像webpack这样的模块打包器，使用你的其他任何东西:

// 使用 ES6 模块
import { Router, Route, Switch } from 'react-router'

// 使用 CommonJS 模块
var Router = require('react-router').Router
var Route = require('react-router').Route
var Switch = require('react-router').Switch

UMD构建也可以使用 unpkg:

<script src="https://unpkg.com/react-router/umd/react-router.min.js"></script>

你可以在 window.ReactRouter找到文库 .
